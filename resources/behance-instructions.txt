Behance Upload Instructions
Updated Oct 2012

0. BI uses Behance to host its portfolio projects, which updates to the website.
1. Log in to your Behance (http://www.behance.net/) account, or create one if you don't have one.
2. Create a new project.
3. Add any relevant images, text, media to describe your project.
4. Make sure Berkeley Innovation is in your Behance inner circle.
5. Add Berkeley Innovation as a co-owner for the project.
6. Once confirmed, your project should now appear on BI's Behance page and website.
